# Old Audio

These files are kept for backup/reference only.

- `legacy_bgm/` contains earlier OGG tracks that are no longer used by gameplay.
- `final_wav_sources/` contains WAV source exports for the active OGG music.

Active music lives in `assets/audio/bgm/final/` and `assets/audio/bgm/end.ogg`. Active sound effects live in `assets/audio/sfx/`.
